1. Общие положения

1.1. Настоящая Политика описывает порядок обработки персональных данных пользователей сервиса EazyClass.

1.2. Используя Сервис, Пользователь соглашается с обработкой персональных данных в соответствии с настоящей Политикой.

2. Какие данные мы получаем

2.1 При авторизации через Telegram

Сервис может получать:

  • Telegram ID
  • имя и фамилию
  • username
  • язык
  • другие данные профиля, предоставленные платформой в соответствии с политикой платформы

2.2 При авторизации через другие платформы

Сервис может получать:

  • уникальный идентификатор пользователя
  • имя
  • данные профиля, предоставленные платформой

2.3 Технические данные

  • IP-адрес
  • cookies
  • данные сессии
  • данные авторизации
  • данные об использовании Сервиса

3. Цели обработки данных

Персональные данные используются для:

  • аутентификации пользователя
  • предоставления функциональности Сервиса
  • отправки уведомлений
  • обеспечения безопасности
  • улучшения работы Сервиса
  • связи с пользователем

4. Хранение и защита данных

4.1. Данные хранятся на серверах, контролируемых Администрацией.

4.2. Применяются технические и организационные меры защиты.

4.3. Данные хранятся только в течение срока, необходимого для целей обработки.

5. Передача данных третьим лицам

5.1. Сервис не продаёт персональные данные.

5.2. Передача возможна только:

  • по требованию закона
  • для работы инфраструктуры Сервиса
  • при использовании сторонних платформ авторизации

6. Cookies и сессии

6.1. Сервис использует cookies и аналогичные технологии для:

  • авторизации
  • безопасности
  • работы интерфейса

6.2. Пользователь может ограничить использование cookies через настройки браузера.

7. Уведомления

7.1. Сервис может отправлять уведомления через Telegram и иные платформы.

7.2. Отправка уведомлений связана с использованием функциональности Сервиса.

8. Права пользователя

Пользователь имеет право:

  • запросить информацию о своих данных
  • запросить исправление данных
  • запросить удаление аккаунта
  • прекратить использование Сервиса

9. Удаление данных

9.1. Пользователь может запросить удаление аккаунта.

9.2. После удаления данные уничтожаются или обезличиваются, если иное не требуется законом.

10. Изменение политики

10.1. Политика может обновляться.

10.2. Актуальная версия всегда доступна в Сервисе.
